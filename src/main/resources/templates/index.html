Tesla Inventory

RESTful API with ability to make get, post, put, and delete requests on Tesla "Inventory".

Routes

GET /teslas Get all Teslas from inventory
GET /tesla/{id} Get Tesla by id
GET /tesla/color/{color} Get all Teslas by color
GET /tesla/model/{model} Get all Teslas by model
GET /tesla/battery/{battery} Get all Teslas by battery
GET /tesla/wheel/{wheelSize} Get all Teslas by wheel size
GET /tesla/interior/{interiorColor} Get all Teslas by interior color
GET /tesla/autopilot/{hasAutopilot} Get all Teslas by autopilot availability
POST /tesla Create a new Tesla
PUT /tesla/{id} Edit existing Tesla by id
DELETE /tesla/{id} Delete Tesla by id
DELETE /teslas Delete all Teslas

Properties

Model S | 3 | X
